C'est quoi SkillCloud ?
Une carte des compétences de la promo, construite par vous.
Voir qui sait quoi

Chaque compétence déclarée apparaît dans le nuage 3D. Plus elle est rare dans la promo, plus elle est colorée. Plus elle suscite d'intérêt, plus elle est grande.

Organiser les ateliers entre pairs

Vous votez pour les compétences sur lesquelles vous voulez progresser. Les plus demandées deviennent des ateliers animés par ceux qui les maîtrisent.

Signaler ce qui manque

Une compétence n'est pas encore dans le nuage ? Vous pouvez la demander. Si quelqu'un dans la promo la possède, il peut y répondre via le formulaire.

Deux liens à retenir :
🔗 Formulaire — pour déclarer vos compétences
🌐 Nuage — pour explorer, voter, demander
Déclarer mes compétences
Via le formulaire — ça prend 3 minutes.
1

Ouvrir le formulaire

Cliquer sur le lien formulaire partagé dans Discord. Il s'ouvre dans votre navigateur, aucune installation nécessaire.

2

Saisir votre prénom

Juste votre prénom — c'est ce qui apparaîtra dans les fiches compétences du nuage. Utilisez toujours le même prénom si vous revenez.

3

Remplir jusqu'à 5 compétences

Pour chaque compétence : choisissez une catégorie, saisissez le nom (les compétences déjà existantes s'autocomplètent), puis choisissez votre niveau.

● Découverte — notions ●● Pratique — expérience ●●● Expert — peut animer
4

Enregistrer

Cliquer sur "Enregistrer mes compétences". Elles apparaissent dans le nuage en quelques secondes, en temps réel pour tout le monde.

Vous pouvez revenir plusieurs fois. Pas de limite — si vous avez oublié une compétence ou changé d'avis sur votre niveau, resoumettez. Les déclarations s'accumulent par prénom.

Voter pour un atelier
Dites sur quoi vous voulez progresser — les plus votés deviennent des ateliers.
1

Se connecter avec Discord

Cliquer sur "Se connecter avec Discord" en haut à gauche du nuage. C'est nécessaire pour voter — ça garantit qu'on ne vote qu'une fois par compétence.

2

Cliquer sur une compétence

La fiche s'ouvre à droite — elle affiche les porteurs, leur niveau, et le nombre de souhaits déjà exprimés.

3

Ajouter à mes souhaits

Cliquer sur "Ajouter à mes souhaits" dans la fiche. Vous pouvez voter pour 10 compétences maximum. Vous pouvez aussi retirer un vote.

Panneau ★ Souhaits (bouton bas droite) — liste toutes les compétences classées par nombre de votes, avec les porteurs et leurs niveaux. Pratique pour voir les sujets les plus demandés d'un coup d'œil.
Demander une compétence
Une compétence manque dans le nuage ? Signalez-la.
1

Ouvrir le panneau Demandes

Cliquer sur le bouton ● demandes en bas à droite du nuage. Le panneau s'ouvre sur le côté.

2

Formuler la demande

Saisir le nom de la compétence souhaitée, une description courte (facultatif), et votre prénom. Pas besoin de choisir une catégorie — c'est le sachant qui le fera quand il répondra.

3

Soutenir une demande existante

Si quelqu'un a déjà formulé une demande qui vous intéresse aussi, cliquez + Je suis intéressé·e et saisissez votre prénom. Les demandes sont classées par nombre d'intéressés.

4

Répondre à une demande

Vous possédez la compétence demandée ? Cliquer sur Je peux apporter cette compétence →. Le formulaire s'ouvre — saisissez la compétence et votre niveau. Une fois déclarée, la demande est automatiquement marquée comme satisfaite.

Demande satisfaite — quand la compétence est déclarée dans le nuage, la demande affiche "✓ Compétence disponible — voir dans le nuage →". Cliquer dessus centre la vue sur la compétence.
